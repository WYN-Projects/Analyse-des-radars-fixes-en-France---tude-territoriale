# Données brutes

Ce dossier contient les fichiers source utilisés dans le projet.

## Fichiers présents

### 1. jeu-de-donnees-liste-des-radars-fixes-en-france-12-2025.csv

Fichier contenant les radars fixes en France.

Colonnes utiles utilisées :
- `numero_radar`
- `type_radar`
- `date_mise_en_service`
- `vma`
- `latitude`
- `longitude`

### 2. communes-france-2025.csv

Fichier contenant les informations sur les communes françaises.

Colonnes utilisées :
- `code_insee`
- `nom_standard`
- `dep_nom`
- `reg_nom`
- `population`
- `latitude_centre`
- `longitude_centre`

## Remarque

Le fichier des communes contient beaucoup plus de colonnes que celles exploitées ici.

Le projet ne conserve que les colonnes utiles à l’analyse territoriale.
